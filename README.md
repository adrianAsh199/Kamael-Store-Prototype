# CashoutOnMe - Tugas Week 5

Aplikasi registrasi Android sederhana yang mengimplementasikan berbagai komponen UI dan validasi.

## Fitur Utama
1. **Form Lengkap**: Input Nama, Email, Password, dan Konfirmasi Password.
2. **Validasi Lanjutan**:
   - Validasi format email.
   - Password minimal 6 karakter.
   - Cek kecocokan password dan konfirmasi password.
   - Validasi real-time (tombol submit hanya aktif jika semua syarat terpenuhi).
3. **Selection Controls**:
   - Pilihan Jenis Kelamin (Radio Button).
   - Pilihan Hobi (Checkbox) dengan minimal 3 pilihan.
4. **Spinner & Dialog**:
   - Dropdown menu (Spinner) untuk memilih pekerjaan.
   - Alert Dialog konfirmasi saat klik tombol Submit.
5. **Gesture Interaction**:
   - Implementasi **Long Press** pada tombol Submit untuk mereset form.

## Teknologi
- Kotlin
- Jetpack Compose (Modern Android UI)
- Material Design 3
